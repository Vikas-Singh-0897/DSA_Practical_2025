This repository covers various fundamental and advanced data structures implemented as part of the practical curriculum.

### Lab Programme List

1. Recursive generation of Fibonacci series up to N terms.
2. Recursive computation of factorial for a given number.
3. Program to insert, delete, and traverse elements in a one-dimensional array. 
4. Program to implement binary search in a sorted array.
5. Program to implement Bubble Sort and display intermediate passes.
6. Program to implement Insertion Sort.
7. Program to implement Merge Sort using recursion.
8. Program to calculate address of an element in a 2D array using row-major and column-major formulas.
9. Program to implement stack using arrays with push, pop, peek, and display operations.
10. Program to implement stack using linked list with dynamic memory allocation.
11. Program to implement queue using arrays with enqueue, dequeue, peek, and display.
12. Program to implement queue using linked list with dynamic memory allocation.
13. Program to implement queue using two stacks
14. Program to create a singly linked list and perform insertion at beginning, middle, and end.
15. Program to delete a node from a singly linked list by value or position.
16. Program to create a binary tree using array representation.
17. Program to perform preorder, inorder, and postorder traversal of a binary tree.
18. Program to build a max heap and perform heapify operation.
19. Program to represent a graph using adjacency matrix.
20. Program to perform Breadth-First Search (BFS) traversal of a graph.
